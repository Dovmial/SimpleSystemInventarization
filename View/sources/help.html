Мини-справка

  • Хранение/загрузка данных осуществляется посредством XML-файла
  • Создать структуру зданий и комнат, добавить оборудование можно через диалоговые окна Редактировать->добавить ...
  • Основное окно поделено на 4 части.
    1. Cписок оборудование в комнате.
    2. Информация о выбранном оборудовании.
    3. Техническое обслуживание.
    4. Обнаруженные проблемы и методы их решения.

  • Редактирование оборудования - двойной клик на оборудовании
  • Поиск оборудования - Редактировать->найти оборудование
  • Добавить запись о техническом обслуживании и проблемах -> контекстное меню оборудования
  • Удаление записи нижних таблиц - через контекстное меню
  • Выбор шрифта отображения информации об оборудовании - через контекстное меню
